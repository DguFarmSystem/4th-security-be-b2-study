   - [ ] 브랜치 전략 정하기 (`main`, `dev`, `feature/`, 등)
   - [ ] 이슈 템플릿 작성

   - [ ] 라벨 설정 (예: `bug`, `enhancement`, `documentation` 등)

3. Spring Boot 초기 세팅
